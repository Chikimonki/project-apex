Native CAN bus decoder with SIMD vectorization. Processes 4 frames simultaneously using hardware vectors.

LuaJIT FFI Bridge

Zero-copy Foreign Function Interface connecting Zig native code to the Lua pipeline orchestrator.

🤖

ROS 2 Integration

Custom F1Telemetry messages with real-time publishing to the ROS 2 ecosystem.

3D Visualization

Real-time rviz2 markers showing speed, RPM, and throttle as animated 3D bars.
